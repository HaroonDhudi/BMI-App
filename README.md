# BMI Calculator App

This is a basic Android app that calculates Body Mass Index (BMI) based on user input.

## Features
- Input weight in pounds
- Input height in feet and inches
- Calculate BMI with a single tap
- Display result in a clean UI


## How to Run
1. Clone the repo
2. Open in Android Studio
3. Build and run on emulator or device
